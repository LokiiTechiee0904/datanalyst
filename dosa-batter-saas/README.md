# DOSA BATTER WhatsApp SaaS

WhatsApp based sales tracker for dosa batter vendors.

Features

- Track 25rs and 35rs batter sales
- WhatsApp chatbot interface
- Daily sales report
- Weekly sales report
- Monthly sales report
- Multi vendor SaaS

Tech stack

Node.js
Supabase
WhatsApp Cloud API
React Dashboard

Deployment

Backend : Render
Database : Supabase
Automation : n8n

Usage

Send "hi" to start.

Menu

1 Enter batter count
2 Start sales
3 Sales summary
4 Daily report
5 Weekly report
6 Monthly report
